# ok:status - DEPRECATED

This command has been replaced by `/ok:resume`.

**Use instead:**
- `/ok:resume` - Resume session with full context restoration
- `/ok:pause` - Save session before stopping work

The new commands provide better conversation context preservation across sessions.
