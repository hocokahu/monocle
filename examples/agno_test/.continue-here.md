<decisions_made>

- Removed explicit `span_name` from methods.py to match other frameworks (auto-generate from package.object.method)
- Tool args passed as kwargs in Agno (`function_call=...`), not positional args - updated entity accessor
- ADK uses `SequentialAgent` (no LLM routing), Agno uses `Team` with `TeamMode.coordinate` (LLM-based)
- Team creates separate traces per member because internal delegation breaks context

</decisions_made>
