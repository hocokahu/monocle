<context>
User asked to extend monocle to support Agno framework (similar to existing ADK, LangChain support). Created metamodel following the established pattern in `instrumentation/metamodel/`. The design principle is: methods.py defines what to instrument, entities/*.py define span types and attribute extraction, _helper.py contains extraction logic.

Agno uses:
- `agno.agent.Agent.run/arun` for agent execution
- `agno.models.base.Model.invoke/ainvoke` for LLM inference
- `agno.models.base.Model.run_function_call/arun_function_call` for tool execution
- FunctionCall has `function: Function` and `arguments: Dict` attributes
</context>
