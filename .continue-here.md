## Key Decisions

- **No `entities/` folder**: Span attributes built inline in `transcript_processor.py`. Entity files were dead code.
- **`SimpleSpanProcessor` not `BatchSpanProcessor`**: Hook is short-lived — batch would lose unflushed spans.
- **Hook at project level**: `monocle_hook.py` lives in `examples/scripts/`, not pip-installed. Like Langfuse hook pattern.
- **Framework package is pip-installed**: `_helper.py` and `transcript_processor.py` are in the `monocle_apptrace` package.
- **Editable install**: `pip install -e apptrace/` so repo changes are reflected immediately.
